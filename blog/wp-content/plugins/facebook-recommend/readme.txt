Very Simple light weight plugin to integrate facebook recommend button on every post

== Description ==

This plugin will allow you to add a facebook recommend button to each of your post, after the post body. Facebook recommend option is an revolutionary invention for social media optimization. Every time, a facebook user click the recommend button, it automatically creates a status post on that user's facebook wall's timeline. So, a huge scope and probabilty of good traffics/targeted visitors are here on using this facebook recommend button.

== Installation ==

1. Upload `facebook-recommend` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are all donee.
